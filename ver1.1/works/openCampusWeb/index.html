多摩美術大学 情報デザインコース
オープンキャンパス2022
HTML　CSS　JAVASCRIPT
オープンキャンパスで情報デザインコースを紹介するWEBサイトを作製しました。
webサイトの動線設計から、ワイヤーフレームを使ったUIデザイン、サイトの雰囲気をまとめたムードボードまで作成し、どのようなサイトにするのかを詳細に決めるところから始めました。最終的には、HTML、CSS、JAVASCRIPTを使って、webサイトとして公開できる状態にまで落とし込みました。
・こだわり
　トップページは、「情報デザインコースの生徒に書いてもらった『インタラクションのイメージ』である矢印の画像をランダムに表示したい」とデザイナーに頼まれていました。しかし、標準のランダム関数を使って表示させるだけではPCとスマホのどちらで見ても見栄えのするグラフィックが出力されませんでした。 　そこで、自分で新たに「どのデバイスからでも、読込ごとに、『いい感じの配置』になるランダム関数」を作りました。この「いい感じの配置」というのはデザインを学んできた人間にしか分からないものだと思います。「美術大学で学んだプログラマー」という個性で、いいものを作ることがと感じています。
