多摩美術大学 情報デザインコース
オープンキャンパス2022
HTML　CSS　JAVASCRIPT
　多摩美術大学のオープンキャンパスで情報デザインコースを紹介するWEBサイトを制作しました。今回の制作は、パンフレットやポスター、PR動画など、様々な媒体で展開されました。そのため、他のチームが制作する作品の雰囲気の雰囲気に添えるよう、グラフィックデザイナーと何度も検討を重ねて見た目や動きを作りました。特にトップページの制作に力を入れました。トップページは、当初「情報デザインコースの生徒に書いてもらった『インタラクションのイメージ』である矢印の画像をランダムに表示したい」とデザイナーに頼まれていました。しかし、標準のランダム関数を使って表示させるだけではPCとスマホのどちらで見ても見栄えのするグラフィックが出力されませんでした。そこで、自分で新たに「どのデバイスからでも、読込ごとに、『いい感じの配置』になるランダム関数」を作りました。この「いい感じの配置」というのはデザインを学んできた人間にしか分からないものだと思います。美術大学にいるプログラマーとしての個性を出すことで、いいものにすることができたと感じた作品です。
