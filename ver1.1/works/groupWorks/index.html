LUMA
ADOBE XD
１年次に製作した「LUMA」は、コロナ禍での新たな出会いの場を提供するスマートフォン向けアプリケーションです。
　大学でのグループワークでサービスをコンセプトから考え、製作しました。
同じ目的を持った仲間を手軽に集める機能にフォーカスし、アプリの機能を設計しました。画面遷移の途中の画面なども作り、ADOBE XDを使用して動くプロトタイプとして完成させました。
・制作背景
　当時はコロナ１年目で、私たちは大学に友人があまりいな状態でした。近くの店や映画館にいくのも、昼食を食べるのも全ての行動が１人で、誰でもいいから一緒に遊べる人が欲しい！という思いから生まれました。
制作時期：2020年（大学1年）
制作期間：半年
制作人数：3人
役割：企画、ファシリテーション、設計
IDQ
FIGMA HTML CSS JAVASCRIPT
２年次に製作した「IDQ」は、情報デザインについて紹介するサイトです。
　大学でのグループワークでサービスをコンセプトから考え、製作しました。
webサイトの動線設計から、ワイヤーフレームを使ったUIデザイン、サイトの雰囲気をまとめたムードボードまで作成し、どのようなサイトにするのかを詳細に決めるところから始めました。最終的には、HTML、CSS、JAVASCRIPTを使って、webサイトとして公開できる状態にまで落とし込みました。
・こだわり
情報デザインというものはとても広範囲に及んでいて、まだまだ新しい、未知の領域を含んでいます。情報デザインを宇宙のように見立て、無限大の可能性を秘めるこの分野について、できるだけ具体的に、身近なところから紹介し、説明しています。
制作時期：2021年（大学2年）
制作期間：半年
制作人数：4人
役割：企画、ファシリテーション、設計、実装
